# learning_earthporn
Using machine learning to determine if an image will make it big time on r/earthporn.

Software needed:
- python3.7
- Python Packages:
-- pandas
-- numpy
-- matplotlib
-- opencv
-- sciket-learn
- python IDE (spyder or pycharm works well)
- Git
-- Git UI is recommended (Gitkraken is a good one)
- anaconda

Other things needed:
- github account
